# AI ML Hybrid Fraud Detection Project

This repository contains the code and evaluation scripts for a hybrid machine learning pipeline designed for fraud detection.

## Dataset
The raw datasets (Data set 1 to Data set 5) are not included in this repository due to size limitations.

**Kaggle Dataset Link:** [INSERT KAGGLE LINK HERE]

## Project Structure
- `hybrid_model_pipeline.py`: Main execution script for the hybrid models.
- `compute_advanced_metrics.py`: Script for calculating performance metrics.
- `requirements.txt`: Project dependencies.
- `*.xlsx` / `*.csv`: Evaluation results and performance summaries.

## Getting Started
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Download the datasets from the Kaggle link above and place them in the corresponding folders.
4. Run the pipeline: `python hybrid_model_pipeline.py`.
